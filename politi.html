Política de Privacidade

Esta Política de Privacidade tem como objetivo esclarecer aos usuários como suas informações são coletadas, utilizadas, armazenadas, protegidas e compartilhadas no GeoFurion, conforme as diretrizes da Lei Geral de Proteção de Dados Pessoais (LGPD - Lei nº 13.709/2018).

1. Dados Coletados

Coletamos os seguintes tipos de dados pessoais, mediante consentimento:

  • Identificação: Nome completo, CPF (opcional), e-mail, telefone.
  • Dados de localização: Coletados via GPS para funcionalidades específicas do app.
  • Dados técnicos: IP, tipo de dispositivo, sistema operacional, navegador, data e hora de acesso.

2. Finalidade do Uso dos Dados

Utilizamos os dados para:

  • Identificar e autenticar usuários.
  • Fornecer funcionalidades de rastreamento e segurança.
  • Enviar comunicações relevantes (como alertas e atualizações).
  • Melhorar o desempenho e segurança da aplicação.
  • Atender obrigações legais e regulatórias.

3. Bases Legais para o Tratamento

O tratamento de dados é realizado com base em:

  • Consentimento: dados processados com a autorização do titular.
  • Execução de contrato: para prestação do serviço contratado.
  • Obrigação legal: quando necessário para cumprir obrigações regulatórias.
  • Legítimo interesse: quando o uso dos dados não viola os direitos dos titulares.

4. Armazenamento e Segurança

Seus dados são armazenados em servidores protegidos com:

  • Criptografia AES-256 e tráfego seguro via HTTPS (TLS 1.3).
  • Ambientes monitorados 24/7, com firewall e controle de acesso.
  • Backups periódicos e criptografados.
  • Auditorias e testes de segurança contínuos.

5. Compartilhamento de Dados

Nenhum dado é vendido ou compartilhado com terceiros para fins comerciais. Compartilhamentos limitados podem ocorrer com:

  • Autoridades legais, mediante requisição formal.
  • Fornecedores contratados para fins técnicos (ex: hospedagem), que seguem cláusulas de confidencialidade.

6. Direitos do Titular dos Dados

Você pode, a qualquer momento, exercer os seguintes direitos:

  • Confirmação da existência de tratamento.
  • Acesso aos seus dados.
  • Correção de dados incompletos ou desatualizados.
  • Anonimização ou exclusão de dados.
  • Portabilidade dos dados a outro fornecedor.
  • Informação sobre compartilhamentos realizados.
  • Revogação do consentimento.

7. Uso de Cookies

Cookies são utilizados para:

  • Armazenar preferências do usuário.
  • Melhorar a navegação e performance do site.
  • Realizar análises estatísticas (com dados anonimizados).

Você pode configurar seu navegador para bloquear cookies a qualquer momento.

8. Serviços de Terceiros

Alguns recursos podem integrar serviços de terceiros (ex: Google Maps). Esses serviços têm suas próprias políticas de privacidade, as quais recomendamos que você consulte.

9. Retenção dos Dados

Seus dados são mantidos apenas pelo tempo necessário para cumprir as finalidades descritas, exceto quando a retenção for exigida por lei.

10. Alterações na Política

Esta política pode ser atualizada periodicamente. A versão mais recente estará sempre disponível neste endereço com a data da última modificação.
